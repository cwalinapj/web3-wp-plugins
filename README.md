# Web3 WordPress Plugins

A collection of WordPress plugins for Web3 and community engagement.

## Plugins

### 1. Contributor Bounties

**Purpose**: Attract writers to publish posts about specific topics by offering bounties/rewards for quality content.

**Location**: `/contributor-bounties/`

**Features**:
- Admin can create bounty campaigns with specific requirements
- Set topic/keywords, content requirements (word count, sources, originality)
- Configure payouts (fixed or tiered)
- Set maximum winners and deadlines
- Contributors submit drafts via frontend submission form
- Optional GitHub PR integration for quality control

See the [Contributor Bounties README](contributor-bounties/README.md) for detailed documentation.

## Installation

1. Clone this repository
2. Copy the desired plugin folder to your WordPress `/wp-content/plugins/` directory
3. Activate the plugin through the WordPress admin panel
